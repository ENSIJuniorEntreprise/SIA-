# Dossier `src/assets`

Utilisez ce dossier pour stocker les photos et images utilisées par l'interface frontend.

Structure recommandée :

- `src/assets/images/` : images générales (logos, photos, icônes)

Conseils :

- N'ajoutez pas de fichiers volumineux au dépôt si possible ; utilisez un stockage externe pour les gros médias.
- Gardez des noms de fichiers descriptifs et sans espaces, par exemple `logo-sia.png`.
